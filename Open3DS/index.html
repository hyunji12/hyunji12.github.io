Abstract

Understanding 3D semantics of a scene is a fundamental problem for various scenarios such as embodied agents. While NeRFs and 3DGS excel at novel-view synthesis, previous methods for understanding their semantics have been limited to 2D understanding: their segmentation results are 2D masks and their ground truth is language embeddings anchored at 2D pixels. This paper revisits the problem setting to pursue the true 3D understanding of a scene modeled by NeRFs and 3DGS as follows. 1) We directly supervise the 3D points to learn language embedding field. It achieves state-of-the-art accuracy without relying on multi-scale language embeddings. 2) We transferred the learned language field to 3DGS, achieving the first real-time rendering speed without sacrificing training time or accuracy. 3) We introduce a 3D querying and evaluation protocol for assessing the reconstructed geometry and semantics together.

Method Overview

We mask and crop every object in multi-view images, then pass them through the CLIP encoder to create scale-free ground truth embeddings. Then we apply pointwise semantic loss to supervise the sampled point embeddings in langauge fields. Furthermore, the learned language fields can be transferred into 3DGS for faster rendering speeds. (Open3DS 3DGS) For querying each 3D and 2D segmentation, we calculate the relevancy score with the given text using the sampled point embeddings and the rendered language embeddings. Lastly, our 3D evaluation protocol measures the 3D segmentation performance both in reconstructed geometry and semantics.
